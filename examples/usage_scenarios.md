# FitTrack Pro 使用场景示例

本文档展示了 FitTrack Pro 的完整使用场景和流程。

## 📖 场景一：新用户首次使用

### 步骤1：设置个人资料

进入"设置"页面，填写以下信息：

```
昵称：张三
性别：男性
出生日期：1990-05-15
身高：175 cm
当前体重：75 kg
目标体重：70 kg
活动水平：中度活动 (每周3-5天)
```

### 步骤2：配置 AI 功能（可选）

在设置页面底部，输入 DeepSeek API 密钥：
```
API 密钥：sk-xxxxxxxxxxxxx
```

### 步骤3：记录初始体重

进入"记录"页面 → "体重"标签：
```
日期：2024-01-01
体重：76.5 kg
体脂率：18.5%
肌肉率：35.2%
备注：新年第一天，开始记录
```

### 步骤4：记录第一天的饮食

**使用 AI 智能记录：**

进入"记录"页面 → "饮食"标签 → 选择"AI 智能记录 🧠"

**早餐：**
```
输入：一碗燕麦粥加香蕉和一杯咖啡
```

**午餐：**
```
输入：150g鸡胸肉配糙米饭和西兰花
```

**晚餐：**
```
输入：三文鱼配红薯和绿色沙拉
```

**加餐：**
```
输入：一杯希腊酸奶
```

### 步骤5：记录运动

进入"记录"页面 → "运动"标签：
```
日期：2024-01-01
类型：跑步
时长：30 分钟
消耗热量：300 kcal
```

### 步骤6：记录日常指标

进入"记录"页面 → "日常指标"标签：
```
日期：2024-01-01
睡眠：7.5 小时
饮水量：2000 ml
心情：8/10
压力等级：3/10
```

### 步骤7：查看仪表盘

进入"仪表盘"页面，查看：
- ✅ AI 健康教练建议
- ✅ 今日热量摄入和消耗
- ✅ 体重趋势图
- ✅ BMI 状态
- ✅ 健康雷达图

## 📖 场景二：日常使用流程

### 每日早晨

1. **记录体重**（可选，建议每周2-3次）
   - 进入"记录" → "体重"
   - 输入今日体重和体脂率

2. **记录早餐**
   - 进入"记录" → "饮食"
   - 使用 AI 智能记录或手动输入

### 每日中午

1. **记录午餐**
   - 进入"记录" → "饮食"
   - 记录午餐内容

### 每日晚上

1. **记录晚餐**
   - 进入"记录" → "饮食"
   - 记录晚餐内容

2. **记录运动**（如果有）
   - 进入"记录" → "运动"
   - 记录运动类型和消耗

3. **记录日常指标**
   - 进入"记录" → "日常指标"
   - 记录睡眠、饮水、心情、压力

4. **查看仪表盘**
   - 进入"仪表盘"
   - 查看今日数据概览和 AI 建议

## 📖 场景三：减重目标追踪

### 目标设定

在"设置"页面设置：
```
当前体重：75 kg
目标体重：70 kg
活动水平：中度活动
```

### 数据记录

**第一周（2024-01-01 至 2024-01-07）：**

- 每日记录饮食（控制热量在 1800-2000 kcal）
- 每周3次运动（跑步、力量训练）
- 每周2次体重记录
- 每日记录日常指标

**第二周（2024-01-08 至 2024-01-14）：**

- 继续记录饮食和运动
- 查看体重趋势图，评估进展
- 根据 AI 建议调整饮食

**持续追踪：**

- 每周查看"分析"页面
- 查看营养成分分布
- 查看健康相关性分析
- 根据数据调整计划

## 📖 场景四：增肌目标追踪

### 目标设定

在"设置"页面设置：
```
当前体重：70 kg
目标体重：75 kg
活动水平：高度活动
```

### 数据记录重点

1. **高蛋白饮食**
   - 每日蛋白质摄入目标：150-180g
   - 使用 AI 智能记录确保营养均衡

2. **力量训练记录**
   - 记录每次力量训练
   - 记录消耗的热量

3. **体重和体脂率**
   - 每周记录体重和体脂率
   - 关注肌肉率变化

4. **日常指标**
   - 确保充足睡眠（7-9小时）
   - 记录心情和压力水平

## 💡 最佳实践

### 1. 数据记录频率

- **体重**：每周2-3次（同一时间、同一条件）
- **饮食**：每日记录（最好餐后立即记录）
- **运动**：每次运动后记录
- **日常指标**：每日睡前记录

### 2. 数据准确性

- 使用 AI 智能记录提高效率

- 定期检查数据一致性

### 3. 目标设定

- 设定现实可行的目标
- 分阶段设定小目标
- 定期评估和调整

### 4. 数据分析

- 关注营养成分分布
- 分析健康相关性

## ⚠️ 常见问题

### Q: 忘记记录某天的数据怎么办？

A: 可以在"记录"页面选择日期，补录之前的数据。

### Q: AI 解析的营养成分不准确怎么办？

A: AI 解析后可以手动调整数值，确保准确性。

### Q: 如何查看历史数据？

A: 在"仪表盘"和"分析"页面可以查看历史趋势和统计数据。

### Q: 数据会丢失吗？

A: 数据存储在本地数据库 `fittrack.db` 文件中，建议定期备份。
