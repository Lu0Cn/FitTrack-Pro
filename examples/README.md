# FitTrack Pro 示例数据和使用案例

本文件夹包含 FitTrack Pro 项目的示例数据和使用案例，帮助您快速了解和使用应用。

## 📁 文件说明

### 数据示例
- `sample_user_profile.json` - 用户资料示例
- `sample_weight_logs.json` - 体重记录示例
- `sample_food_logs.json` - 饮食记录示例
- `sample_exercise_logs.json` - 运动记录示例
- `sample_daily_metrics.json` - 日常指标示例

### 使用案例
- `ai_food_logging_examples.md` - AI 智能饮食记录使用示例
- `usage_scenarios.md` - 完整使用场景示例
- `import_sample_data.py` - 导入示例数据的 Python 脚本

## 🚀 快速开始

### 方法一：使用 Python 脚本导入示例数据

```bash
python examples/import_sample_data.py
```

### 方法二：手动导入

参考 `usage_scenarios.md` 中的说明，手动在应用中输入示例数据。

## 📝 使用说明

1. **查看示例数据**：打开 JSON 文件了解数据格式
2. **参考使用案例**：查看 `usage_scenarios.md` 了解完整使用流程
3. **尝试 AI 功能**：参考 `ai_food_logging_examples.md` 中的示例输入

## ⚠️ 注意事项

- 示例数据仅供演示和学习使用
- 导入示例数据会覆盖现有数据库（如果使用脚本导入）
- 建议在测试环境中使用示例数据
