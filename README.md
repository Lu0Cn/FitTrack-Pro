# FitTrack Pro 💪

一个功能强大的健康追踪应用，基于 Streamlit 构建，支持 AI 智能分析和多语言界面。

## 📋 项目简介

FitTrack Pro 是一个全面的健康与健身追踪系统，帮助用户记录和管理体重、饮食、运动以及日常健康指标。应用集成了 DeepSeek AI，提供智能饮食分析和个性化健康建议。

## ✨ 主要功能

### 📊 仪表盘（Dashboard）
- **AI 健康教练**：基于每日数据生成个性化健康建议
- **关键指标展示**：摄入热量、运动消耗、总消耗、热量结余
- **体重趋势图**：可视化过去30天的体重变化，显示目标体重线
- **BMI 状态**：实时显示 BMI 值和健康状态（偏瘦/正常/超重/肥胖）
- **健康雷达图**：综合评估睡眠、心情、压力和活动水平
- **周度热量平衡**：展示一周内的热量摄入与消耗对比

### 📝 记录（Logger）
- **体重记录**：记录体重、体脂率、肌肉率，支持备注
- **饮食记录**：
  - 🤖 **AI 智能记录**：使用自然语言描述饮食，AI 自动解析营养成分
  - ✍️ **手动记录**：手动输入食物名称、热量、蛋白质、碳水、脂肪
- **运动记录**：记录运动类型、时长、消耗热量
- **日常指标**：记录睡眠时长、饮水量、心情评分、压力等级

### 📈 分析（Analysis）
- **营养成分分析**：今日宏量营养素（蛋白质、碳水、脂肪）分布饼图
- **健康相关性分析**：睡眠与心情的关系散点图，压力水平可视化
- **活动热力图**：过去30天的运动活动日历热力图

### ⚙️ 设置（Settings）
- **用户资料管理**：昵称、性别、出生日期、身高、当前体重、目标体重、活动水平
- **AI 集成配置**：DeepSeek API 密钥设置
- **多语言支持**：支持中文（简体）和英文切换

## 🛠️ 技术栈

- **前端框架**：Streamlit
- **数据库**：SQLite
- **数据处理**：Pandas
- **数据可视化**：Plotly
- **AI 集成**：DeepSeek API（OpenAI 兼容）
- **语言支持**：中英文双语

## 📦 安装与运行

### 环境要求

- Python 3.7+
- pip

### 安装步骤

1. **克隆或下载项目**
   ```bash
   cd Final
   ```

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

   或者使用批处理文件（Windows）：
   ```bash
   run.bat
   ```

3. **运行应用**
   ```bash
   streamlit run app.py
   ```

   应用将在浏览器中自动打开，默认地址：`http://localhost:8501`

## 🗄️ 数据库结构

应用使用 SQLite 数据库，包含以下表：

- **user_profile**：用户个人资料
- **weight_logs**：体重记录
- **food_logs**：饮食记录
- **exercise_logs**：运动记录
- **daily_metrics**：日常指标（睡眠、饮水、心情、压力）

## 🔑 API 配置

要使用 AI 智能功能（智能饮食记录和健康建议），需要配置 DeepSeek API 密钥：

1. 访问 [platform.deepseek.com](https://platform.deepseek.com) 注册并获取 API 密钥
2. 在应用的"设置"页面中配置 API 密钥
3. 配置完成后即可使用 AI 功能

## 💡 使用说明

### 首次使用

1. 启动应用后，首先进入"设置"页面完善个人资料
2. （可选）配置 DeepSeek API 密钥以启用 AI 功能
3. 选择界面语言（中文/英文）

### 日常使用

1. **记录数据**：在"记录"页面添加体重、饮食、运动和日常指标
2. **查看概览**：在"仪表盘"页面查看今日数据概览和 AI 建议
3. **深度分析**：在"分析"页面查看营养成分分布和健康趋势

### AI 智能记录示例

在"记录"页面的"饮食"标签中，选择"AI 智能记录"模式，可以用自然语言描述饮食：

- 英文示例：`"2 eggs and toast"`
- 中文示例：`"一碗燕麦粥加香蕉和一杯咖啡"`

AI 会自动解析并提取营养成分信息。

## 📊 计算公式

### BMI（身体质量指数）
```
BMI = 体重(kg) / 身高(m)²
```

### BMR（基础代谢率）- Mifflin-St Jeor 公式
- **男性**：`(10 × 体重) + (6.25 × 身高) - (5 × 年龄) + 5`
- **女性**：`(10 × 体重) + (6.25 × 身高) - (5 × 年龄) - 161`

### TDEE（总消耗能量）
根据活动水平乘以相应系数：
- 久坐：1.2
- 轻度活动：1.375
- 中度活动：1.55
- 高度活动：1.725
- 超高活动：1.9
